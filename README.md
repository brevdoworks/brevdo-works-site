# Brevdo Works website

Simple React/Vite landing page for Brevdo Works.

## Local preview

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy on Vercel

1. Upload this folder to a GitHub repository.
2. Go to Vercel.
3. New Project → Import GitHub repository.
4. Deploy.
5. Add your domain in Project → Settings → Domains.
